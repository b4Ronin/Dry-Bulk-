# Dry-Bulk-

Bulk Sack Calculator V1.

## Features

- Offline-capable PWA
- Install to iPhone/Android home screen
- Saves last tank capacity and safe fill level
- Input lock/unlock
- Product selector with stored cubic feet per sack factors
- Automatic calculation:
  - Reserve Capacity = Tank Capacity × (100 - Safe Fill Level)
  - Usable Capacity = Tank Capacity × Safe Fill Level
  - Sacks = Usable Capacity ÷ Product Factor
